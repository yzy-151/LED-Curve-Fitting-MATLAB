# LED 非线性曲线拟合（MATLAB）

一个用于 LED 电压-电流特性建模的 MATLAB 课程实验。脚本比较不同非线性参数下的曲线，并依据给定数据搜索模型参数。

## 功能

- 绘制 `k = 2、3、50` 时的 LED 非线性模型曲线
- 使用二次多项式拟合实测电压-电流数据
- 通过网格搜索估计最优饱和电流 `i_max` 与形状参数 `k`
- 输出拟合误差并绘制最终拟合曲线

## 运行

使用 MATLAB 打开并运行：

```matlab
Light_exp1
```

脚本会生成三幅图，并在命令窗口输出搜索得到的参数及误差。

## 文件

- `Light_exp1.m`：模型计算、数据拟合、参数搜索与绘图

## 说明

这是课程实验代码，数据与参数范围直接写在脚本中，适合用于理解 LED 非线性建模和基础参数拟合流程。
